# C++ Codes 

> A structured and comprehensive collection of C++ programs, covering everything 
> from fundamentals to advanced concepts.

---

##  About This Repository

This repository serves as a complete reference for the C++ programming language.
Whether you are a beginner taking your first steps or an experienced developer 
looking for quick references, this repo has you covered.

All code is written in standard C++ and organized by topic for easy navigation.

---

##  Repository Structure

| Folder | Contents |
|--------|----------|
| `01_Basics` | Variables, Data Types, Operators, I/O |
| `02_Control_Flow` | If-Else, Switch, Loops |
| `03_Functions` | Functions, Recursion, Overloading |
| `04_Arrays_Strings` | Arrays, 2D Arrays, String Manipulation |
| `05_Pointers` | Pointers, References, Memory Management |
| `06_OOP` | Classes, Objects, Inheritance, Polymorphism |
| `07_STL` | Vectors, Maps, Sets, Queues, Stacks |
| `08_File_Handling` | Reading & Writing Files |
| `09_Data_Structures` | Linked Lists, Trees, Graphs |
| `10_Algorithms` | Sorting, Searching, Dynamic Programming |
| `11_Advanced` | Templates, Exception Handling, Multithreading |

##  How to Use

1. **Clone the repository**
```bash
   git clone https://github.com/MonstrousSh4rj/C-Codes.git
```

2. **Navigate to a topic folder**
```bash
   cd C-Codes/06_OOP
```

3. **Compile and run any file**
```bash
   g++ filename.cpp -o output
   ./output
```
